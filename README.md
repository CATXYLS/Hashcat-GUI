# Hashcat-GUI
支持zip、pkzip、rar和7z（需配置perl）<br/><br/>
你需要提前准备<a href="https://github.com/hashcat/hashcat">hashcat</a>、<a href="https://github.com/openwall/john/tree/bleeding-jumbo">john</a>和<a href="https://strawberryperl.com/">perl</a>并放置在程序所在目录下<br/><br/>
暴力破解完成后会生成历史记录和password.txt到压缩包目录下<br/><br/>
(如果系统环境无法识别请下载便携式压缩包并解压到程序目录下)
